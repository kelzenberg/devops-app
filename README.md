# DevOps App

This is a project description.

## Develop the project

Run `yarn install` once to install dependencies.  

- Start TS compiler in watch mode
  ```sh
  yarn dev
  ```

- Lint project
  ```sh
  yarn lint
  # or
  yarn lint:fix
  ```

- Format project
  ```sh
  yarn format
  ```

- Compile project
  ```sh
  yarn compile
  ```
